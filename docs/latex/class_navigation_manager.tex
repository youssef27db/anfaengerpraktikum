\doxysubsection{Detailed Description}
Der \doxylink{class_navigation_manager}{Navigation\+Manager} ist für das Laden von Leveln und das Spawnen des Spielers verantwortlich.

Der \doxylink{class_navigation_manager}{Navigation\+Manager} ist ein Singleton, der in der Haupt-\/\+Szene platziert wird und von anderen Skripten verwendet wird, um Level zu laden und den Spieler zu spawnen.
